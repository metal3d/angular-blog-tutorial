Page d'accueil

Voilà la page principale.

J'ai créé le service "blog" via 

yo angular:service blog

Cela a créé app/services/blog.js. Allez voir le fichier.

Ce service retourne un objet contenant deux méthodes pour aller chercher des données sur un serveur web.

J'ai entrée un post de blog "manuellement" dans le fichier db/blog.json.

Enfin, dans le fichier "app/scripts/controllers/main.js" j'ai ajouté la dépendance au service "blog" et j'apppelle getPosts() 

{{ posts | json }}

Attention, $resource retourne des promesses ! mais angular va interpréter le résultat au fur et à mesure que les promesses sont résolues.

Ici, on va chercher le post 1, via getPost(1)

{{ firstPost | json }}

On peut passer à step 3

git checkout -f step3 && docker-compose restart
